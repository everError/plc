## 1. what-is-plc.md

### PLC란?

**Programmable Logic Controller**의 약자로, 산업 현장에서 기계나 공정을 자동으로 제어하는 산업용 컴퓨터입니다.

### 구성요소

| 구성요소  | 역할                     |
| --------- | ------------------------ |
| CPU       | 프로그램 실행, 연산 처리 |
| 전원부    | AC/DC 전원 공급          |
| 입력 모듈 | 센서, 스위치 신호 수신   |
| 출력 모듈 | 모터, 밸브, 램프 제어    |
| 통신 모듈 | 외부 장치와 데이터 교환  |
| 메모리    | 프로그램 및 데이터 저장  |

### 동작 방식

```
[입력 읽기] → [프로그램 실행] → [출력 쓰기] → (반복)
```

이 사이클을 **스캔(Scan)**이라 하며, 보통 수~수십 ms 단위로 반복됩니다.

### 일반 PC와의 차이

| 항목   | PLC              | PC             |
| ------ | ---------------- | -------------- |
| 목적   | 실시간 제어      | 범용           |
| 환경   | 고온, 진동, 먼지 | 사무실         |
| OS     | 없음 또는 RTOS   | Windows, Linux |
| 안정성 | 24시간 연속 운전 | 재부팅 필요    |
